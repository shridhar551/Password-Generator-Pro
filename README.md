# 🔐 Password Generator Pro

A modern and simple Password Generator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Generate strong random passwords
- Select password length (4–50 characters)
- Include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*)
- Password strength indicator
- Copy password to clipboard
- Responsive design
- Fast and lightweight
